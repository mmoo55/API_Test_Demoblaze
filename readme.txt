Este archivo contiene los pasos para realizar la ejecución de las pruebas. Se recomienda el uso de IntelliJ IDEA
Siga a detalle cada uno de los pasos siguientes:
    1. Importar el archivo.
    2. Para ejecutar el test, se lo puede realizar de diferentes maneras:
       - Si se encuentra utilizando IntelliJ IDEA, presione las teclas Mayús+F10
       - Si esta utilizando IntelliJ IDEA, dirigirse al directorio /Project/API_Test/src/test/java/KarateRunner.java,
         a continuación presione el ícono "run" (flecha verde)
    3. Para acceder al reporte que ofrece el framework Karate, puede realizarlo de diferentes maneras:
       - Ingresar al link que aparece en el log.
       - Dirigirse al directorio /API_Test/target/karate-reports/karate-summary.html, donde encontrará el archivo ".html"
    4. Una vez dentro del reporte, podra dirigirse a los diferentes "features" y observar los diferentes "Scenario" que
       fueron ejecutados.
